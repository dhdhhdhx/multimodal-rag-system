一、数据的表示
------------
1. 原码：最高位为符号位，0正1负，其余为绝对值
   例如：+5 = 00000101，-5 = 10000101

2. 反码：
   - 正数：反码与原码相同
   - 负数：符号位不变，其余位取反
   例如：-5的反码 = 11111010

3. 补码：
   - 正数：补码与原码相同
   - 负数：反码加1
   例如：-5的补码 = 11111011
   计算机中整数默认使用补码存储

4. 移码：补码的符号位取反，常用于浮点数阶码

5. 浮点数表示（IEEE 754）：
   浮点数 = (-1)^S × 1.M × 2^(E-127)
   - S：符号位（1位）
   - E：阶码（8位，偏移量127）
   - M：尾数（23位）

二、CPU组成
---------
CPU = 运算器 + 控制器

运算器组成：
- ALU（算术逻辑单元）：执行算术运算和逻辑运算
- 累加寄存器（AC）：存放运算结果
- 状态条件寄存器（PSW）：存放运算状态标志（OF/SF/ZF/CF）
- 数据缓冲寄存器（DR）：暂存数据
- 地址寄存器（AR）：存放地址

控制器组成：
- 程序计数器（PC）：存放下一条指令地址
- 指令寄存器（IR）：存放当前正在执行的指令
- 指令译码器（ID）：对指令进行译码
- 时序发生器：产生时序信号
- 操作控制器：产生各种操作控制信号

三、寻址方式
----------
1. 立即寻址：操作数直接在指令中
2. 直接寻址：指令中直接给出操作数的有效地址
3. 间接寻址：指令中给出操作数地址的地址
4. 寄存器寻址：操作数在寄存器中
5. 寄存器间接寻址：操作数的地址在寄存器中
6. 基址寻址：基址寄存器内容 + 偏移量
7. 变址寻址：变址寄存器内容 + 形式地址
8. 相对寻址：PC内容 + 偏移量

四、指令流水线
------------
流水线周期 = 最长段的时间

流水线执行时间 = 第一条指令完整执行时间 + (n-1) × 流水线周期
               = k × t + (n-1) × t  （k为段数，t为最长段）

流水线吞吐率 = n / (k×t + (n-1)×t)，当n→∞时，TP = 1/t

流水线加速比 = 不用流水线时间 / 用流水线时间

五、存储系统
----------
存储层次结构（速度快→慢）：
寄存器 → Cache → 主存 → 外存

Cache：
- 命中率 h = 命中次数 / (命中次数 + 未命中次数)
- 平均访问时间 = h × tc + (1-h) × tm
  （tc为Cache访问时间，tm为主存访问时间）

映射方式：
- 直接映射：主存块只能映射到Cache的固定行
- 全相联映射：主存块可映射到Cache任意行
- 组相联映射：分组直接映射，组内全相联

替换算法：
- FIFO：先进先出
- LRU：最近最少使用（最常用）
- LFU：最不经常使用
- 随机替换

六、总线
------
总线分类：
- 数据总线：传输数据
- 地址总线：传输地址信息，决定寻址空间
- 控制总线：传输控制信号

系统总线标准：
- ISA：16位，8MHz
- PCI：32/64位，33MHz
- PCIe：串行，支持热插拔

七、可靠性与校验码
---------------
1. 海明码：发现双错，纠正单错
   - 校验位位数 k 满足：2^k ≥ n + k + 1
   - 校验位放在 2^i 位置

2. CRC循环冗余校验码：
   - 生成多项式确定校验位数
   - 通过模2除法计算校验位

3. 奇偶校验：只能检测奇数个错误，不能纠错
